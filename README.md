# ⚽ FIFA World Cup Live Hub

**FIFA World Cup Live Hub** is a dynamic football-themed website created as a **college case study project**. The website provides information about FIFA World Cup matches, players, and teams while also offering real-time match updates and access to official platforms where users can watch matches live.

The goal of this project is to demonstrate **frontend web development skills** using HTML, CSS, and JavaScript while building an interactive and visually engaging sports information platform.

## 🌍 Overview

The website acts as a central hub for football fans to explore match information, player details, team data, and upcoming fixtures. It also includes real-time features such as live match status, current time display, and match countdown timers to make the experience more dynamic and engaging.

The interface is designed with a modern football-themed layout inspired by stadium visuals and sports dashboards.

## ✨ Features

* **Live Match Section** – Displays ongoing matches with team names, score updates, and match stage.
* **Upcoming Matches** – Shows scheduled matches with kickoff time and countdown timer.
* **Players Section** – Highlights famous football players with photos, positions, and statistics.
* **Teams Section** – Provides information about national teams participating in the tournament.
* **Top Scorers Leaderboard** – Displays players leading the goal-scoring rankings.
* **Watch Live Section** – Allows users to visit official platforms such as the FIFA website or FIFA YouTube channel to watch matches.
* **Photo Gallery** – Shows memorable football moments and player images.
* **Login / Sign Up Page** – A simple authentication interface for users.
* **Real-Time Clock** – Displays the current time on the website using JavaScript.
* **Responsive Design** – Optimized for desktop, tablet, and mobile devices.

## 🛠 Technologies Used

* **HTML5** – Used to structure the website and organize content.
* **CSS3** – Used to design the layout, colors, animations, and responsive design.
* **JavaScript** – Used for dynamic features such as match updates, countdown timers, and interactive elements.
* **Font Awesome** – Used for icons throughout the website.
* **Google Fonts** – Used to enhance typography and visual style.
